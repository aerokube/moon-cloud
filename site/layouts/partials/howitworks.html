How it works

  • You get a completely dedicated cluster in one of supported cloud platforms: AWS, Google Cloud or Microsoft Azure.
  • We always run this cluster in the geographic region near you, so network connectivity is lightning fast.
  • Moon Cloud is running in Kubernetes has autoscaling feature, i.e. shrinks and grows depending on how many browsers you need. Because of that you only pay for computing resources you really consume.
  • You can use all popular browser automation frameworks: Selenium, Playwright, Puppeteer and Cypress simultaneously.
  • Moon Cloud relies on cloud networking features and can access to your workstation, corporate network server or cloud project.
